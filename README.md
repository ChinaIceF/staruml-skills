# StarUML-Skills

Agent-friendly skills for generating and editing UML diagrams in StarUML `.mdj` projects.

## Overview

**StarUML-Skills** is a collection of skills designed to help AI coding agents create, read, and modify UML diagrams inside StarUML `.mdj` files.

The project focuses on automating UML modeling workflows for StarUML.

These skills are intended to guide an agent through the structure of StarUML `.mdj` files and provide reusable instructions for creating consistent, editable UML models.

## Platform Compatibility

This project is designed as a set of skills for **opencode**.

The current structure, naming, and skill descriptions are written with opencode-style agent workflows in mind.

If you want to use these skills with another agent platform, framework, or local LLM environment, you may need to migrate or adapt them manually.

Possible migration work may include:

* Rewriting the skill metadata

## What This Project Is

This project is:

* A collection of reusable agent skills
* A helper layer for working with StarUML `.mdj` files
* A starting point for AI-assisted UML diagram generation
* An experimental automation project
* A reference for building StarUML-oriented modeling agents

## What This Project Is Not

This project is not:

* An official StarUML project
* A complete UML modeling framework
* A replacement for StarUML
* A formally verified `.mdj` manipulation library
* A guaranteed correct implementation of all UML semantics
* A universal UML exchange format tool

StarUML `.mdj` files are StarUML project files. They can store UML models, but they are not the same as standardized UML exchange formats such as XMI.
